# 🎰 Casino Games Platform

A fully functional multi-game casino platform built with Next.js and React. Play multiple games like Blackjack, Mines, Baccarat, Limbo, Dice, Chicken, and Slide.

## Features

- 🃏 **Blackjack** - Classic card game
- 💣 **Mines** - Avoid the mines and cash out
- 🎴 **Baccarat** - Predict player, banker, or tie
- 📈 **Limbo** - Stop before the crash
- 🎲 **Dice** - Pick a number and roll
- 🐔 **Chicken** - Keep playing or cash out
- 🏄 **Slide** - Find the correct tiles

## Tech Stack

- **Frontend**: Next.js 14 + React 18
- **Styling**: Tailwind CSS
- **Storage**: localStorage for balance persistence
- **Deployment**: Ready for Vercel

## Getting Started

### Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

### Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Deploy with one click!

## How to Play

1. Start with $1000 in play money
2. Choose a game from the lobby
3. Place your bet
4. Play the game
5. Win or lose - your balance updates
6. Click "Back to Lobby" to play another game

## Game Rules

### Blackjack
Get closer to 21 than the dealer without going over.

### Mines
Click safe tiles to increase your multiplier. Avoid mines!

### Baccarat
Bet on Player, Banker, or Tie. Highest hand value wins.

### Limbo
Set a target multiplier and hope the value stays below it.

### Dice
Pick a number 1-6 and roll. Match to win 6x your bet.

### Chicken
Keep playing rounds to increase multiplier or cash out.

### Slide
Find the correct tile to advance. One wrong choice ends the game.
